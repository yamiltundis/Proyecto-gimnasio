Sistema web para gestion de un gimnasio. En él se gestionarán:
- Clientes
- Membresias
- Pagos
- Clases
- Reservas
- Control de asistencias
- Roles: administrador/cliente (próximamente)

Tecnologias:
- Frontend: React
- Backend: Node.js, TypeScript, Express, Prisma (próximamente Zod y JWT)
- Persistencia: MySql corriendo en un contenedor Docker (con volumen para no perder los datos)
